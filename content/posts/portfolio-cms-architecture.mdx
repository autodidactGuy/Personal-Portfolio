This project reorganizes a portfolio into a scalable content system.

## Scope

- Static export compatible with GitHub Pages
- Decap CMS editing under `/cms-admin`
- MDX for long-form content
- JSON for structured UI content

## Outcome

The result is a portfolio where authors can change content without editing React components directly.
